03.SQL

什么是SQL？

​ Structured Query Language：结构化查询语言，其实就是定义了操作所有关系型数据库的规则。

每一种数据库操作的方式存在不一样的地方，称为“方言”。

SQL通用语法

  • SQL 语句可以单行或多行书写，以分号结尾。

  • 可使用空格和缩进来增强语句的可读性。

  • MySQL 数据库的 SQL 语句不区分大小写，关键字建议使用大写。

  • 3 种注释

    • 单行注释: – 注释内容 或 # 注释内容(mysql 特有)
    • 多行注释: /* 注释 */

SQL分类

1) DDL (Data Definition Language)

​ 数据定义语言

​ 操作数据库，表

​ 关键字：create, drop,alter 等

2) DML (Data Manipulation Language)

​ 数据操作语言

​ 增删改表中的数据

​ 关键字：insert, delete, update 等

3) DQL (Data Query Language)

​ 数据查询语言

​ 查询表中的数据

​ 关键字：select, where 等

4) DCL (Data Control Language)

​ 数据控制语言(了解)

​ 授权

关键字：GRANT， REVOKE 等

数据库数据类型：

int：整数类型

  • age int,

double:小数类型

  • score double(5,2)

date:日期，只包含年月日，yyyy-MM-dd

datetime:日期，包含年月日时分秒 yyyy-MM-dd HH:mm:ss

timestamp:时间错类型 包含年月日时分秒 yyyy-MM-dd HH:mm:ss

  1. 如果将来不给这个字段赋值，或赋值为null，则默认使用当前的系统时间，来自动赋值

varchar：字符串

  1. name varchar(20):姓名最大20个字符
  2. zhangsan 8个字符 张三 2个字符
