  • 如果file文件后面携带文件名，则查看指定文件的类型

4.文件权限

  1. 读【r】read
  2. 写【w】write
  3. 执行【x】execute

d rwx rwx rwx 分为三组

6.常用命令

文件管理

ls：查看文件

参数 含义
-a 查看文件，包括隐藏文件
-l 以列表方式查看文件，不包括隐藏文件
-h 文件大小以 KB ， MB等计量单位显示出来
-all 以列表方式查看所有文件，包括隐藏文件
通配符 含义
* 文件代表文件名中所有字符
ls ch* 查找以te开头的文件
ls *an 查找以an结尾的文件
？ 代表文件名中任意一个字符
ls ?.c 只找第一个字符任意，后缀为.c的文件
ls a.? 只找只有3个字符，前2字符为a.，最后一个字符任意的文件
ls [abc] abc之间穷举法
ls [a-f] a到f 之间 穷举法
ls [a-f]* 找到从a到f范围内的的任意一个字符开头的文件
ls a-f 查找文件名为a-f的文件,当“-”处于方括号之外失去通配符的作用

>：输出重定向命令

Linux允许将命令执行结果重定向到一个文件，本应显示在终端上的内容保存到指定文件中。

如：ls > test.txt ( test.txt 如果不存在，则创建，存在则覆盖其内容 )

|：管道

  • 连接两个命令

  • ls | more

  • -s 表示创建软链接

  • 注意：如果软链接文件和源文件不在同一个目录，源文件要使用绝对路径，不能使用相对路径。

cat：查看或合并文件内容

  • 可以同时查看多个文件

  • cat a.txt b.txt

cat a.txt c.txt > d.txt
参数 含义
^a 搜索以a开头的内容
ke$ 搜索以ke结尾的内容
[Ss]igna[Ll] [ ] 中的内容，符合条件都显示出来
. 代表任意字符

wc：计算文件行数或字数

常用选项说明：

选项 含义
-l 统计行数
-w 统计字数。一个字被定义为由空白、跳格或换行字符分隔的字符串。
-c 统计字节数
-m 统计字符数。这个标志不能与 -c 标志一起使用。
  • 一般直接这样使用
