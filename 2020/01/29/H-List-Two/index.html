checked checked 定义选择控件中默认被选中的项
maxlength 正整数 控件允许输入的最多字符数

7.input标签新增属性

属性 属性值 含义
placeholder 由用户自定义 框中 浅淡 提示文字 输入后自动消失
autofocus autofocus 或 无 光标自动定位到输入框中 方便直接输入
multiple 无 多文件上传
autocomplete on ，off 或 无 自动记录输入过的信息
required 无 规定某框为 必填项
accesskey 某个字母 ALT + 字母，快速定位到某框

8.emded标签

属性 属性值 描述
height 设置嵌入内容的高度。
width 设置嵌入内容的宽度。
src url 嵌入内容的 URL。

9.audio标签

属性 值 描述
autoplay autoplay 自动播放
controls controls 如果出现该属性，则向用户显示音频控件（比如播放/暂停按钮）。
loop loop 重复播放
muted muted 静音
preload auto，metadata 或none 规定当网页加载时，音频是否默认被加载以及如何被加载。
src URL 规定音频文件的 URL。
  • 若属性和其属性值相同，则可以直接写 属性名

10.video标签

loop loop 重复播放
muted muted 静音
preload auto，metadata 或none 规定当网页加载时，视频是否默认被加载以及如何被加载。
src URL 规定视频文件的 URL。
