CSS样式规格

概念： Cascading Style Sheets 层叠样式表

  • 层叠：多个样式可以作用在同一个html的元素上，同时生效

好处：

  1. 功能强大
  2. 将内容展示和样式控制分离
    • 降低耦合度。解耦
    • 让分工协作更容易
    • 提高开发效率

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明:

  1. 选择器通常是您需要改变样式的 HTML 元素。
  2. 每条声明由一个属性和一个值组成。
  3. 属性是您希望设置的样式属性。每个属性有一个值。属性和值被冒号分开。

0.1 字体样式属性

属性值 含义
font-size 字体大小，px为单位
font-family 字体，例如 宋体
font-weight 字体粗细
font-style 字体是否倾斜

常用技巧：

  1. 网页中普遍使用14px+

  2. 使用偶数的数字字号

  3. 各种字体之间用 逗号隔开

  4. 字体名称中 若包含特殊符号或为中文名称时，则需要这样

  5. 选择多种字体时，名称为中文，一般放到后面

  6. 字体选择 键入时，尽量 使用 unicode ，一般字体 只选择 宋体和Microsoft YaHei

font-weight

值 描述
nomal 默认值
bold 定义粗体字符。

font-style

值 描述
normal 默认值。
italic 斜体
  • 一般文字很少使用 斜体

0.3 CSS外观属性

3.1 CSS颜色格式

line-height：行间距

  • 单位：em，px 和 % ，em表示一个字的宽度
  • 一般情况下，行距 比字号大 7，8个px即可

text-align：水平对齐方式

  • left，right 和 center

text-indent：首行缩进

letter-spacing : 字间距

word-spacing：单词间距

颜色半透明

text-shadow:文字阴影

盒子模型：控制布局

margin：外边距

padding：内边距

默认情况下内边距会影响整个盒子的大小

box-sizing: border-box; 设置盒子的属性，让width和height就是最终盒子的大小

float：浮动

  • left
  • right
