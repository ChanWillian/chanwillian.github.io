什么是HTML?

HTML 是用来描述网页的一种语言。

  • HTML 使用标记标签来描述网页
  • HTML 文档包含了HTML 标签及文本内容
  • HTML文档也叫做 web 页面

web标准：

结构 XML,XHTML
样式 CSS
行为 DOM ,ECMAScript

标签属性

1.标签可以有多个属性 ，属性必须写在开始标签中，位于标签名后面

2.属性之间不分先后顺序，标签名与属性，属性与属性之间均以空格隔开

3.任何标签的属性都有默认值，省略该属性则取默认值

图片标签

属性 属性值 描述
src URL 图片的路径
alt 文本 图片不能显示时的替换文本
title 文本 鼠标悬停时的内容
width 像素 设置图片宽度，一般只改高度就行
height 像素 设置图片高度，一般只改高度就行
border 数字 设置图片边框的宽度

链接标签
