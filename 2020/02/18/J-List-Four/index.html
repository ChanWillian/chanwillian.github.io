1.多线程

  • 单核：实际上 是单线程，实现程序速度快，从宏观上看，实现了多线程
  • 多核：真正意义上的多线程
  1. 进程内并发执行的代码段。并发：即同时。
  2. 线程之间共享内存。
  3. 多线程的意义：创造灵活响应的桌面程序。

线程：

  • 每个运行着的线程对应一个stack。
  • 应用程序至少有一个线程（主线程）。

进程

  1. 运行时的应用程序。
  2. 进程之间的内存不是共享的，是独占的。
  3. 进程间通信时用的socket ( 套接字 ) 。

2.Thread

创建多线程类

继承——》Thread

  • 使用单核运行此多线程

  • 调用线程类时，调用 start 函数，而不是 run 函数，run 函数 是由 CPU 调用。

  • 此程序比较简单，进行优化

多线程实质：

  1. 在 main stack 中 ，创建 多线程对象。
  2. 调用 start 函数，t1 , t2 开始 实现线程。
  3. 在 t1 , t2 线程，分别条用 run 函数。

Thread.yield ( ) 方法

  • 让当前线程让出 CPU 抢占权：这是个瞬间的动作。

作业题

Thread.sleep( )方法

  • 让当前线程休眠指定毫秒数。
  • 放到 try——catch 语句中

Thread.join( )方法

  • 当前线程等待指定的线程结束后才能继续进行
  • 放到 try——catch 语句中
  • cmd 窗口下，输入 regedit ，打开注册表。
